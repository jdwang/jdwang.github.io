关于设备分辨率

关于设备分辨率

1. 分辨率分为物理分辨率和逻辑分辨率

2. 物理分辨率是硬件显示的分辨率（不可变）

3. 逻辑分辨率是软件（系统）用来解析的分辨率（可变）

4. 不同分辨率下图像大小发生变化，指的是逻辑分辨率

5. 非retina iPad（iPad2及以下）物理分辨率1024x768 逻辑分辨率 1024x768

6. retina iPad（new iPad及以上）物理分辨率2048×1536 逻辑分辨率 1024x768

前端方面

1. css中的像素指的是逻辑像素

2. 适配某个设备的分辨率指的是逻辑分辨率

3. css3 media query 检测的是设备的逻辑分辨率

4. -webkit-device-pixel-ratio检测的是物理分辨率与逻辑分辨率的比值（物理分辨率中的长除以逻辑分辨率的长，或者是两个宽相除）；对应的js属性是window.devicePixelRatio；这个值retina屏苹果产品是2，非retina屏是1，安卓中有1.5的情况

  • 同样15’的mpb换了屏幕的那台比起普通的15‘mbp看网页宽度变小两边空白变多和文字变小，是因为它在增大了物理分辨率的同时支持了1680x1050的逻辑分辨率；

  • 13’rmbp文字、视频显示效果比普通13’mbp清晰得很多但是没有网页、文字大小改变，是因为只有物理分辨率变大了而逻辑分辨率没有变；
